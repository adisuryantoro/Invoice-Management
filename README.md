# Application Invoice Management #

This Application can be managed invoices and connect with various payment methods.
Supported payment methods : 

* Virtual account bank :
    * Bank BNI
    * Bank CIMB
    * Bank BSI
        
* E-wallet :
    * OVO
    * GoPay
        
* QR Payment :
    * QRIS
